# 🧿 Neon Aura AR – Hand Tracking Experience

An advanced real-time **AI + WebGL + MediaPipe hand tracking AR system** that creates magical neon effects, particles, gestures, and interactive physics using your webcam.

---

## ✨ Features

- 🖐️ Real-time hand tracking (MediaPipe)
- ⚡ Pinch detection → lightning + shockwaves
- 🌈 Neon glow particle system
- 🧙 Magic shield effects on open palm
- 🎵 Dynamic Web Audio synthesis
- 🌌 Matrix rain background reacting to motion
- 🎨 Multiple visual themes (Cyberpunk, Galaxy, Ocean, etc.)
- 📷 Laptop webcam support (auto-detection)

---

## 🚀 Tech Stack

- HTML5
- CSS3 (Glassmorphism UI)
- JavaScript (Vanilla)
- MediaPipe Hands
- Web Audio API
- Canvas 2D Graphics
